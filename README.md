# DrifterBossGrab

Allows Drifter to grab and bag anything

## Mod Compatibility

This mod is compatible with [Risk of Options](https://thunderstore.io/c/riskofrain2/p/Rune580/Risk_Of_Options/) for in-game configuration.
<br>[ProperSave](https://thunderstore.io/package/KingEnderBrine/ProperSave/) for save/loading bagged objects.

## Wiki

See [Wiki](https://thunderstore.io/package/pwdcat/DrifterBossGrab/wiki/4854-home/) for more information of all the features. Formula examples are included.
